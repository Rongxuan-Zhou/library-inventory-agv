# Source Code Reference

This directory contains representative IEC 61131-3 Structured Text (ST) code that illustrates the
key control logic patterns used in the project. These are **reconstructed reference implementations**
based on the compiled project analysis — the actual CODESYS project source is proprietary.

The code demonstrates:
- PLCopen motion control patterns (MC_ function blocks)
- DS402 EtherCAT drive interfacing
- HTTP REST API client implementation in ST
- State machine design for industrial AGV control
- TCP socket communication for sensor integration

All code follows IEC 61131-3 syntax and is compatible with CODESYS V3.5 SP17+.
